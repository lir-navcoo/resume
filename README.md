| 类别 | 技术 |

## 📦 本地开发

```bash
# 克隆仓库
git clone https://github.com/lir-navcoo/resume.git
cd resume

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 🔐 密码修改说明

如需修改访问密码，请编辑源码中的 `CORRECT_PASSWORD` 常量：

```ts
// src/App.tsx
const CORRECT_PASSWORD = 'your-password'
```

修改后重新 build 并部署即可。
